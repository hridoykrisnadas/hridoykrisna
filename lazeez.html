Github Link

Project Description

Lazeez is a Online Resturant management webapp. The project develop by technology stack comprising HTML, CSS, and JavaScript for the front-end, Java Spring Boot Framework for the back-end, IntelliJ IDEA Ultimate as the integrated development environment (IDE), and MySQL for the database.

The main features of this project is:
  • Customer Signup, Login- Any customer can be signup and login in to this webapp.
  • Menu- Anyone can see the active menu of this resturant. Customer can add to cart his choosable menu item and put order from cart item.
  • My Account- Customer can be see his previous and current order details from this option. Customer can update order and payment status from here.
  • Employee Login/Logout- Employee can login to the application or logout from the application for management the resturant.
  • Food Menu- Employee can add menu item from this option and can be update menu item is active or not.
  • Order- Employee can view Order Details and update Order status for preaparing food or other status.
