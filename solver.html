Solver- Office Management App

Developing Tools
  • Framework: Android Studio
  • Language: Java
  • Database: Firebase, MySQL (Rest API's)
This is a secret app of a company, so others information are not be shown.

Project Description

A complete Trading office management solutions app based on Android.

The main features of this project is:
  • Schedule Management- Admin can make employee schedule for a day. Employee also make Check In/Out with location against this schedule.
  • Tada Management- Employee can apply Tada against the schedule.
  • Cash Requisition Management- Employee can apply for Cash Requisition for office purpose expense.
  • Leave Management- Employee can apply and manage leave from this feature.
  • Car Management- Employee can request for car schedule, Admin can be approve schedule. Driver manage add stoppage and start & stop the schedule with locations. Driver can get Overtime Information with time calculations. Driver can apply for Car Expense from App.
  • Quotation Management- Employee can be request for quotation from App with collaborate with another stock app of company.
  • Workplace Conversion- Employee can doing messaging from this app.
