Project Description

Car Management Systme is complete web based application where a corporate office maintain his cars & drivers easily. The project develop by technology stack comprising HTML, CSS, and JavaScript for the front-end, Java Spring Boot Framework for the back-end, IntelliJ IDEA Ultimate as the integrated development environment (IDE), and MySQL for the database.

The main features of this project is:
  • Employee Registration- A admin can be register employee, driver from this option.
  • Employee Login/Logout- Employee can login to the application or logout from the application.
  • Car Registration- A admin can register car with necessary information from this option.
  • Car Request- All employee can be request for car schedule for his necessary need with pick & drop address and schedule time.
  • Car Request Approve- A admin have permission for approve the car request and assign driver.
  • Car Schedule Update- After complete the schedule, Admin can be update the start time and stop time as per as driver logbook. After updated, the driver duty time have calculated as per his schedule time & he get the duty bill as per as policy.
  • Driver Duty Payment- A admin can be entry payment information to Driver as per his/her duty bill.
  • Car Expense- A admin can be upload car expense with vouchers image like CNG bill, Tool fee, Octen bill etc.
  • Car Expense Payment- A admin can be entry the payment to driver information of car expense
